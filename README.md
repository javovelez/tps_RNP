## Metodología de entrega

Una vez que terminaste de resolver el laboratorio, seguí estos pasos antes de entregarlo:

1. **Ejecutar el notebook completo** desde la primera celda hasta la última, de manera que todas las celdas de código muestren sus resultados. En Colab esto se hace con **Entorno de ejecución > Ejecutar todo** (o `Ctrl+F9`).

2. **Descargar el archivo** desde el menú de Colab: **Archivo > Descargar > Descargar .ipynb**.

3. **Subir el archivo descargado** al aula virtual del curso, en la tarea correspondiente a ese laboratorio.

> El notebook debe entregarse con todas las celdas ejecutadas y sus salidas visibles. Un notebook sin ejecutar o ejecutado parcialmente no se considera completo.
